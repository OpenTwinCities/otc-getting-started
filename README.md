# Who are we?

Open Twin Cities fosters an inclusive community of civic collaboration.  We have three areas of focus:

- Open Data
- Open Software
- Engaged Citizenship

Open Twin Cities is a part of the [Code for America Brigade network](https://brigade.codeforamerica.org/).
